# Feedback Form

ReactJS

## Requirements

- Node
- virtualenv

## Installation

```bash
npm install
```

## Usage

To launch the application.

```bash
npm run start
```

To build an executable. The output binary will be produced in the `dist` directory.

```bash
npm run build
```

To start a development server (only for testing frontend code).

```bash
npm run dev
```
